
## 주요 기술 개념
- **App-Level Access Control** (앱 레벨 접근 제어 / 평가 전용 모드)
- **`appAccessControl` 필드**: 'full' 또는 'member_only' 값
- **round-app vs round-member** 애플리케이션 분리
- **미들웨어 레벨 리다이렉트**로 접근 제어
- **RBAC 계층**: Owner > Admin > Manager > User
